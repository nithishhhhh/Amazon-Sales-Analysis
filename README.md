#  Amazon Sales Analysis using SQL

##  Project Overview

This project analyzes Amazon sales data using SQL to extract business insights related to revenue, customer behavior, product performance, and geographic trends.

The dataset contains **4,570 orders** and was analyzed using SQL queries involving aggregation, grouping, conditional logic, and sorting.

##  Objectives

The goal of this project was to answer real business questions such as:

* How much revenue is generated?
* Which products sell the most?
* Which locations generate the highest sales?
* Which fulfillment method performs better?
* Where are high-value customers located?

---

##  Dataset Features

The dataset includes:

* Order ID
* Date
* Status
* Fulfillment method
* Category
* Quantity
* Amount
* City & State
* Promotion info
* B2B flag

##  Business Questions Solved

###  Sales Overview

* Total number of orders
* Total revenue generated
* Total quantity sold
* Cancellation percentage
* Average order value
* Sales date range

###  Fulfillment Analysis

* Which fulfillment method handles most orders?
* Which fulfillment type has higher average order value?

###  Product Analysis

* Which category generates highest revenue?
* Which category sells most quantity?

###  Location Analysis

* Which states generate highest revenue?
* Which cities place most orders?

###  Advanced Insights

* Which category performs best in each state?
* Which cities have highest average order value?
